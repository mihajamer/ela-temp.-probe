The contents of ela-temp.-probe are programs meant to read BLE advertisements made by ela temperature probes. They are written for python and MATLAB.
Test-broadcast-connection is a short python code which outputs the name, device data and manufacturer data thats being broadcasted by an ela device. The only change that has to be made is the device address, which is different for every probe and can be obtained via any NFC reader on a PC or phone.
Main-program-temp+voltage is meant to read and display temperatures that are live broadcasted by an ela T-PROBE. On one of the first readings it also displays the current battery voltage of the device. CTRL-C can be used to stop the program.
Main-program-with-MATLAB is the same program as Main-program-temp+voltage with changes which make it viable to be used with a MATLAB script. The readings that it gets are being written in a local txt file, which has to be in the same path as the python program.
MATLAB-dynamic-graph is a MATLAB script that draws a live graph based on the temperature readings of the probe and also outputs the voltage. Its important to note that the python program and txt file its creates have to be correctly saved and the path has to be disclosed in MATLAB. The graph it creates uses REAL TIME readings that the python outputs and is continous until stopped. When stopped, MATLAB also stops the python program and deletes the contents of its txt file.
